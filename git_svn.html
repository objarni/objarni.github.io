How I use git inside svn repositories

.gitignore

ignore old binary folders and similar (speed up git dev flow) ignore svnup add .gitignore to svn global ignore pattern

Working on local branches

git branch, git checkout, git commit

Fetching changes from svn repo

svnup, rebase conflicts (git add, git rebase --continue)

Committing changes to svn repo

checkout master, merge branch, svn commit
